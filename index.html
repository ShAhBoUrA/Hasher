Easy Hash

MD5, SHA256, SHA512 (SALTED) hashing lib

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Easy Hash lib.

Small, Easy to use hashing library, providing PLAIN / SALT hash ability with (MD5, SHA256, SHA512). Hash verification feature is also implemented.
Entirely built using C#

// Compute SHA256 PLAIN hash
var sha256Hash = hasher.SHA256ComputeHash("helloWorld");
// Verify SHA256 PLAIN hash
hasher.SHA256VerifyHash("helloWorld", sha256Hash);

// Compute SHA256 SALT hash
sha256Hash = hasher.SHA256SaltComputeHash("helloWorld");
// Verify SHA256 SALT hash
hasher.SHA256SaltVerifyHash("helloWorld", sha256Hash);

Target Framework

.Net 2.0 and above.

Projects involved

  1. Hasher, dll implementing various hashing algorithms
  2. HasherDemoTest, console application testing the Hasher project
